Projetos-Frontend-Senac

Este repositório contém projetos de front-end desenvolvidos com HTML5, CSS3 e JavaScript durante o curso técnico de informática do Senac com o professor Cristiano Maia. O objetivo desse repositório é praticar e aprimorar habilidades em desenvolvimento web, explorando diferentes aspectos dessas tecnologias.

Conteúdo

  • Projetos básicos e intermediários em HTML5
  • Estilização avançada com CSS3
  • Implementação de interatividade com JavaScript
