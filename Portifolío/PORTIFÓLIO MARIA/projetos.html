OBJETIVO

Atualmente, estou focada em vários projetos de programação que abrangem diferentes áreas e tecnologias. Um dos principais projetos é um aplicativo móvel para gerenciamento de tarefas, que estou desenvolvendo utilizando html+css+JavaScript. Este projeto está na fase de implementação das funcionalidades principais, com um foco especial na usabilidade e na interface do usuário.

Além disso, estou trabalhando em um projeto pessoal de desenvolvimento web usando Django e Vue.js, que visa criar uma plataforma para compartilhamento de recursos educacionais. Neste caso, estou na fase inicial de planejamento e prototipagem, definindo os requisitos principais e elaborando o design da aplicação.

Por fim, estou contribuindo em um projeto de código aberto relacionado à análise de dados utilizando Python e Pandas, onde estou colaborando com outros desenvolvedores para melhorar a eficiência e a escalabilidade das análises realizadas.

No geral, estou comprometida em aprender novas tecnologias e melhorar minhas habilidades como desenvolvedora, enquanto trabalho para entregar projetos de alta qualidade e impacto positivo.
